# Mean, Median, Mode Imputation

These are simple statistical methods to handle missing data.

---

## Mean Imputation
- Used for numerical data
- Replace missing values with column mean
- Affected by outliers

---

## Median Imputation
- Used for numerical data
- Replace missing values with median
- Robust to outliers

---

## Mode Imputation
- Used for categorical data
- Replace missing values with most frequent value

---

## Advantages
- Simple and fast
- Preserves dataset size

---

## Disadvantages
- Reduces variance
- Can distort relationships

---

## One-liner
Mean, median, and mode imputation replace missing values using summary statistics.
