# KNN Imputer

KNN Imputer fills missing values using the K-nearest similar rows.

---

## How it works
- Find K nearest neighbors
- Use mean (numerical) or mode (categorical)

---

## Advantages
- Preserves relationships
- More accurate than mean imputation

---

## Disadvantages
- Computationally expensive
- Requires feature scaling

---

## When to use
- Small to medium datasets
- Correlated features

---

## One-liner
KNN Imputer replaces missing values using similar observations.
