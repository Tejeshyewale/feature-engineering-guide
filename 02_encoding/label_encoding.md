# Label Encoding

Label Encoding is a technique used to convert categorical values into numerical values.

---

## How it works
Each unique category is assigned a numeric label.

Example:
- Low → 0
- Medium → 1
- High → 2

---

## When to use
- Categorical data is **ordinal**
- Order of categories matters

---

## Advantages
- Simple and fast
- Requires less memory

---

## Disadvantages
- Can introduce false order if used on nominal data

---

## Common use cases
- Education level
- Rating scale
- Size (Small, Medium, Large)

---

## One-liner
Label encoding converts categorical values into ordered numerical labels.
